Victoria's Reflection

Purpose of the HTML5 semantic tags is to help organize the content by providing more context around what the type of content the tags hold. The order browsers display elements is from the top to the bottom unless it is modified in a css file. I learned that I can customize sublime to help me be more efficient and make less mistakes such as using snippets. The concept that was most difficult was trying to distinguish the different HTML5 semantic tags and the nuances on how to use them.

Steph's Reflection

HTML5 semantic tags provide more context to the content. Browsers rpocess and display elements in a top down order. Sublime’s ability to create customized snippets is an extremely powerful tool that will help save time. I can see myself creating snippets for common layout structures and scripts. It is still not 100% clear to me when we need to use certain HTML5 semantic tags and when it’s totally overkill.
